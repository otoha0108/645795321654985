タイアップ情報

アニメ「サンダー3」ED

主人公の妹「ふたば」と「ぷーすけ」が
夜中にテレビを見ている映像

※そのテレビには「もう一つの世界」が映っている

この作品世界観と、ヒトリエの皆さん特有のネットカルチャー感との間に高い親和性を感じました。

楽曲コンセプト

「しゅるれりら」とは

考える前に、創造が起きてしまっている状態

子供が「シュルレアリスム」という言葉を
意味も分からず、聞き間違えて、舌ったらずに発音しているイメージ

重要：

意味はない / 定義できない / でも音として口に出したくなる

→ 考える前に、創造が起きてしまっている状態そのもの

歌詞全文

あぁ、目が覚めたら
違う誰かに変わってないかな
誰もが認める天才とか
完全無欠のヒーローとか
ちっぽけな夜の中 夢を見る
涙の海に潜り込んだら
まっさらな宇宙を泳ぎに行こう
案外世界も救えるさ

誰かの夢のかなたへ
知らない星のかなたへ
しゅるれりら しゅるれりら
あの宙もこえてゆく

ひしゃげた柵を乗りこえ
割れた鏡を踏みこえ
しゅるれりら しゅるれりら
その声に飛んでゆく

昼行灯の唐変木の木偶の坊でも
世界を全部ひっくり返す夜を待ってる

サウンド方向性

■ 基本コンセプト

バンドサウンドではなく、電子音・シンセサイザー中心の構成

■ 音色・音響イメージ

• 8bit/16bit的なレトロゲームサウンド
• ピコピコ・ピロリロといった明るい電子音
• チップチューン的な質感（ただし古臭くならないように）
• グリッチ・ノイズを効果的に配置

■ 効果音の活用

• 昔の漫画/アニメ的なSE（ビョイーン、ドカーン等）
• ゲームのカーソル音・決定音
• テレビのノイズ・砂嵐
• リモコンの"ピッ"音（冒頭必須）

サウンド方向性（続き）

■ 構成・展開

• イントロ：ノイズ → リモコン音 → 静寂から立ち上がる
• Aメロ：比較的シンプル、ボーカルを聴かせる
• サビ：一気に賑やかに、音数を増やす
• アウトロ：余韻を残しつつフェードアウト

■ 目指す質感

• 明るく賑やかだが、どこか不穏
• ポップだが、完全には整理されていない
• 子供向けアニメ的な軽快さと、大人が感じる違和感の共存
• ネット/ゲーム世代に刺さるノスタルジー

重要：大衆性は保ちつつ、構造のどこかに"ズレ"や"異物"を残してください。
ただし、それを説明する必要はありません。

必ず残したい要素

冒頭の音（必須）

• ノイズ音
• リモコンの"ピッ"という音

これらは楽曲のコンセプト上、必須の要素です。
「テレビをつける=もう一つの世界へ入る」という導入を音で示します。

構造に残す"異物"

大衆向けの入口は最大限に広く保ちながら、
構造のどこかに違和感・ズレ・不穏さを仕込んでください。

※ただし、それを説明する必要はありません。

ターゲット感覚（二重構造）

疲れた大人

もう思考する体力がない
正しさや判断を求められることに疲弊している
AIや時代の加速に適応しようとして、気力を失っている

今の子供たち

まだ意味や正解を理解しなくていい存在
音・リズム・動きで世界を受け取る年齢
これからさらに加速する社会に入っていく

同じ音楽が、別のレイヤーで受け取られる状態を目指しています。

依頼事項まとめ

お任せしたいこと

• 電子音・効果音の選定と配置
• サビの賑やかさと不穏さのバランス
• ネットカルチャー的な"ズレ"の演出
• 全体的な音響デザインとミックス
• レトロゲーム的質感の現代的アップデート

守ってほしいこと

• 冒頭のノイズ + リモコン音は必須
• 1分30秒（TV尺/スポット用）
• 説教・啓蒙的にならないこと
• 大衆性は保ちつつ、どこかに"刺さる刃"を残す

最後に

この曲が伝えたいのは、
「考えなくていい、理解できなくていい、正しくなくていい」
「でも、創造することだけはやめないでほしい」

これは主張ではなく、状態です。
思考を通らずに起きる、外に出す行為としての創造。

何卒よろしくお願いいたします。
